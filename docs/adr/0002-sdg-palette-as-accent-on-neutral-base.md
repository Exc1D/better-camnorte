# Visual identity: SDG palette as rotating accents on a neutral base

BetterCamNorte's identity is a neutral, near-monochrome base — surfaces, text, and a single
provincial primary for global chrome — with the UN SDG 17-colour palette used only as rotating
accents in a defined set of slots (hero, section seams, stat highlights, category tags). We
rejected both a fully SDG-saturated UI (17 colours competing per page) and a purely restrained
institutional palette with no SDG colour at all.

The neutral base guarantees the contrast, hierarchy, and authority that official government data
needs, while the contained SDG accents give the site its optimistic, mission-aligned character.
The accent rotation is thematic, not a strict taxonomy: the colours signal civic optimism and add
variety, they do not encode which specific SDG a page belongs to. We accept that the festive
character is therefore expressed in bounded slots rather than across the whole interface — a
deliberate trade of maximal vibrancy for trust and readability.
